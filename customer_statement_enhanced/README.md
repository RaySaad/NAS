# Customer Statement Report Enhanced

## Description
Professional and modern customer statement report for Odoo 18 with enhanced design and features.

## Features
- Modern gradient header design
- Company branding with logo
- Color-coded debits (red) and credits (green)
- Running balance calculation
- Aging analysis (0-30, 31-60, 61-90, 90+ days)
- Summary boxes for quick overview
- Professional table formatting
- VAT and complete customer details
- Clean, easy-to-read layout

## Installation
1. Copy the module to your Odoo addons directory
2. Update the apps list
3. Install "Customer Statement Report Enhanced"

## Usage
1. Go to Contacts → Customers
2. Select a customer
3. Click Print → Customer Statement (Enhanced)

## Requirements
- Odoo 18.0
- account module
